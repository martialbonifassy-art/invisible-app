I ALREADY OWN A JEWEL

Access the whisper linked to your wooden piece.

Each jewel has its own ID, engraved on a card, a label or on the packaging. Enter this ID to listen to the whisper currently associated with your object.

The ID usually looks like BIJOU001, BIJOU002, etc. Use the exact letters and numbers you see.
If the ID does not work or if the jewel seems silent, you can contact the Atelier so that we can check its configuration and, if necessary, recharge its whispers.
